# CSS 属性值定义语法

### 语法构成

> 关键字 + 数据类型 + 符号

### 符号类型

- 字面符号
  | 符号 | 名称 | 描述 |
  | --- | --------- | ------------------------------- |
  | , | 并列分隔符 | 用来分割数个并列值，或分割函数的参数值 |
  | / | 缩写分隔符 | 用来分隔一个值的多个部分，在 CSS 缩写中用于分离类型相同但属于不同 CSS 属性的值，以及用在部分 CSS 函数中 |

- 组合符号
  | 符号 | 名称 | 描述 |
  | -------| ---- | ------------------------------- |
  | ( )空格 | 并列 | 符号为普通空格字符，表示各部分必须出现，同时需要按顺序出现 |
  | && | 与组合符 | 表示各部分必须出现，出现顺序可随机 |
  | \|\| | 或组合符 | 表示各部分至少出现一个，可以不按顺序出 |
  | \| | 互斥组合符 | 表示各部分只出现一个 |
  | [] | 方括号 | 将各部分进行分组以绕过上面几个符号的优先规则，因此方括号的优先级最高 |

- 数量符号
  | 符号 | 名称 | 描述 |
  | -------| ---- | ------------------------------- |
  | | 无数量符号 | 出现一次 |
  | \* | 星号 | 出现任意次数 |
  | + | 加号 | 出现一次或多次 |
  | ? | 问号 | 出现一次或不出现 |
  | \{A, B\} | 花括号 | 出现 A~B 次 |
  | # | 井号 | 出现一次或多次，但必须用,逗号分隔 |
  | ! | 感叹号 | 该符号多出现在组合符号方括号的后面，表示当前分组必须产生一个值 |
