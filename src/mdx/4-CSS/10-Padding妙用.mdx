## Padding 妙用 -- 分隔符 && 固定比例 && Icon

- 利用了行内元素设置 padding 不影响布局，实际发生渲染但不影响行高的特性，可实现分隔符
- 块级元素 padding 百分比值，水平和垂直方向都是根据宽度计算的，设置百分比可实现宽高比例固定
