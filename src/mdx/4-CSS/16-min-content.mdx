# width: min-content;

首选最小宽度 min-content 是相对 content-box 而言的，内容类型不同，首选最小宽度也不同；

- 可替换元素：img、video、button 等，元素自身的宽度首选最小宽度
- CJK 文字：不存在标点符号时，单个文字即首选最小宽度。文字中存在避尾标点和避头标点时，需包含文字和标点。避尾和避头各浏览器实现不同。
- 非 CJK 文字：最小字符单元为首选最小宽度，所有连续的英文字母、数字和标点都被认为是一个字符单元，直到遇到中断字符。
  > 中断字符: 连字符(-)、问号、大中小前括号, 一般短横线和英文问号只有当后面是字母或数字的时候才会中断字符单元，前 括号则只有当前面不是字母和数字的时候才会中断字符单元。不同浏览器有具体的不同实现。
